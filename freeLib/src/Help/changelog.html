История версий

7.4.0

  • НОВОЕ: Для книг epub добавлено отображение Серии книги и Номера Серии, если они есть.
  • НОВОЕ: Для epub книг добавлено чтение Названия книги из аттрибута name='calibre:title_sort' тега meta, которое будет использоваться, если в описании epub отсутствует тег dc:title, служащий для основного описания Названия книги.
  • ИСПРАВЛЕНИЕ: Исправлен баг, когда не отображались данные Аннотации, Обложка и время создания fbd файлов (djvu, doc и т.д.), когда эти файлы размещены в zip архиве.
  • ИСПРАВЛЕНИЕ: Исправлен баг, когда для epub книг для одного и того же Автора вместо одной записи формировалось несколько.

7.3.0

  • НОВОЕ: ВАЖНО: В данной версии программы используется база данных, таблицы и поля которой не совпадают с прежней базой данных. При запуске, по подтверждению пользователя, будет удалена старая и создана новая база.
  • НОВОЕ: Сделана обработка NickName Автора книги. Если у Автора нет ни Фамилия, ни Имени, ни Отчества, но есть NickName, то этот nick попадает в список Авторов. Такой Автор больше не оказывается в списке Unknown Author.
  • ИСПРАВЛЕНИЕ: Исправлен баг, когда база данных не переносилась в указанную в Настройках папку, потому что в ней уже была база данных.

7.2.0

  • НОВОЕ: Для Windows добавлена возможность сборки win32 (x86) версий программы.

  • НОВОЕ: В меню Справка добавлен пункт вывода информации о Qt фреймворке.

  • НОВОЕ: Если в epub нет тэга genre, или этот тэг - пустой, то книга попадает в список Жанров 'Неотсортированное'.
  • НОВОЕ: Вкладка по работе с Группами книг: Создание, переименовывание, удаление группы из списка групп. Очистка группы от книг (удаление всех книг из группы). Удаление выделенной книги из группы. Две группы ('Избранное', 'Читаю' и 'К прочтению') являются заблокированными: их нельзя удалить. Через контекстное меню можно удалять или устанавливать иконки Группам, а также - восстанавливать иконки по умолчанию для заблокированных Групп.
  • НОВОЕ: Быстрые клавиши для простановки/снятия цветной метки.
  • НОВОЕ: Быстрые клавиши для простановки/снятия рейтинга.

  • УДАЛЕНО: Удален визард создания библиотеки, так как он избыточен из-за наличия Менеджера Библиотек, и сопровождать и синхронизировать по возможностям оба инструмента не имеет смысла.

  • УЛУЧШЕНО: Данные о выделенной книге заносятся на панель метаданных и в случае, когда сама панель опущена вниз разделителем и не видна. Иначе происходит несоответствие между выбранной книгой и метаданными на панели, если ее 'поднять' сплиттером.
  • ИСПРАВЛЕНИЕ: Исправлен старый баг, когда считывался из zip архива не fbd файл, а сама книга, что приводило к тому, что метаданные книги не определялись, и Автор становился Неизвестным, Жанр - Неотсортированным и т.д.
  • ИСПРАВЛЕНИЕ: Исправлен старый баг: После создания новой библиотеки не обновлялись списки Авторов и Серий для этой библиотеки.
  • ИСПРАВЛЕНИЕ: Обложка для книг в формате epub теперь масштабируется под размеры, заданные в Настройках.
